# mindTrack

A next-generation productivity and discipline system that combines planning, focus enforcement, activity tracking, and behavior analytics.

## Monorepo layout

- `apps/web`: planning UI and analytics dashboard
- `apps/api`: backend API, business rules, and data layer
- `apps/desktop-agent`: desktop tracking and focus enforcement agent
- `docs`: planning docs and delivery checklist

## Current milestone

Day 1 foundation:

- Workspace bootstrap
- Core domain schema for planning MVP
- Initial implementation checklist for daily execution
